Buổi 13 : Git reset - Hủy bỏ commit
Hủy bỏ những commit mới, quay lại commit cũ

git reset --soft <commit id> : 

- di chuyển HEAD về vị trí commit.
- trạng thái của stage và tất cả sự thay đổi của file được giữ nguyên.

git reset <commit id> : di chuyển HEAD về vị trí commit reset, vẫn giữ tất cả thay đổi của file, nhưng loại bỏ các thay đổi stage

git reset --hard : di chuyển con trỏ HEAD về vị trí commit reset và các loại bỏ tất cả sự thay đổi của file, stage.
